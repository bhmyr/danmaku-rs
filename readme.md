# B站弹幕记录

- 代码 **90%** 由 **deepseek** 使用 **rust** 生成

## 使用流程

### 首次使用

- 修改 **livers.toml** 添加 **B站** 主播

- 终端运行程序，扫码登录，开始运行

- 登录信息记录在 **auth.toml**

- 弹幕信息记录在 **data/*.csv**

- 发送 **reload** 到端口 **24349** 重新加载 **livers.toml**

- 发送 **stop** 到端口 **24349** 停止程序

### 自定义

- 使用 **sh** 或者 **ps1** 脚本控制启动、重载、停止

- 配置文件 **config.toml**，修改需要记录的信息，存储方式与位置，控制端口
